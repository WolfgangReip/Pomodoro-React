# Log-Yourself-In

* *Version*: 1.0.0
* *Authors*: Surya Hendricks, Wolfgang Reip

Description
----
User account creation and edition connected to a MySQL database. Complete form validation. 

Technologies
----
*PHP*, *Docker*, *MySQL*

What can you do?
----
* Create an account
* Connect to your user space
* Manage your user informations
* Disconnect
* Delete account
